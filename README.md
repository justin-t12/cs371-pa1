# CS371 Project 1


To build the combination client/server program run:

``` shell
make -k all
```

or run:

``` shell
gcc -pthread -o pa1_skeleton pa1_skeleton.c
```


For a debug build, run:
``` shell
make -k debug
```

Usage:

``` shell
./pa1_skeleton <server|client> [server_ip server_port num_client_threads num_requests]
```
